# AI Multi-Agent Content Automation System

## Overview
This project is a multi-agent AI system that automates enterprise content lifecycle.

## Features
- Content Generation
- Compliance Checking
- Issue Fixing
- Optimization
- Localization
- Multi-channel Distribution

## How to Run

1. Install dependencies:
pip install streamlit

2. Run:
python -m streamlit run app.py

## Tech Stack
- Python
- Streamlit

## Architecture
Multi-agent pipeline:
Planner → Generator → Compliance → Fix → Optimizer → Localization → Distribution
